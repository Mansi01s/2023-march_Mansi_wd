Quetion 7:-What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements?

Ans:-Difference between the ‘id’ attribute and the ‘class’ attribute

No. Id attribute Class attribute
1. The ID name starts with the "#" symbol followed by a unique
name assigned to it.		 The class name starts with the "." followed by class name assigned
to it.
2. Only one ID selector can be attached to an element. Multiple class selectors can be attached to an element.

Example:-

ID attribute:

Mansi Savaliya

Class attribute:

Delhi

Delhi is the capital of India.

London

London is the capital of England.
