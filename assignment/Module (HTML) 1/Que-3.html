Quetion 3:-What are void elements in HTML?

Ans:-A void element must have a start tag but not have an end tag, Void elements can have attributes.

List of void Elements:

  1. area - clickable, defined area in an image
  2. br - line break
  3. col - column in a table
  4. hr - horizontal rule
  5. img - image
  6. input - field where users enter data
  7. base - specifies a base URL from which all links base
  8. link - links an external resource to the document
  9. meta - provides information about the document
  10. etc....
