Quetion 3:- What is the difference between CSS2 and CSS3?

Ans:- Difference between CSS2 and CSS3

No. CSS2 CSS3
1. CSS2 stand for Cascading Style Sheets Level 2 CSS3 stand for Cascading Style Sheets Level 3
2. CSS2 doesn't support the Border-Box property. CSS3 supports the Border-Box property.
3. Property of CSS2 is Background, Border, Color, Font, Float, Display, etc... Property of CSS3 is Border-Box, Transition, Text-decoration, etc....
