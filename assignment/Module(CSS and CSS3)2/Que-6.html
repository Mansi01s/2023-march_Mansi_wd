Quetion 6:-How can the background color of an element be changed?

Ans:- Use the CSS Background or Background-color Property changed the background color of an element. After giving Property name give a color name or '#' value or linear-gradient value or radial gradient value or rgb value or rgba value or hsl value or hsla value.

Example:-

1. color name
2. '#' value
3. linear-gradient value
4. radial gradient value
5. rgb value
6. rgba value
7. hsl value
8. hsla value
