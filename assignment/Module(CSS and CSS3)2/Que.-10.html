Quetion 10:-Why should background and color be used as separate properties?

Ans:- Background and color be used as separate properties because background property used for give a background color and color proprety used for give a text color.

Syntax:

  • background: black;
    • Or
  • Background-color: gray; (Apply in Background)
  • color: white; (Apply in Text)
