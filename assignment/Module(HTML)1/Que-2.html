Quetion 2:-What are tags and attributes in HTML?

Ans:-HTML tags: HTML Tags are building blocks of HTML Page.
         HTML attributes: All HTML elements can have attributes, Attributes provide additional information about elements.

Example:-

HTML tags:

  • <b>
  • <br>
  • <em>
  • <img>
  • <link>
  • etc....

HTML attributes:

  • rel
  • src
  • type
  • id
  • name
  • etc....
