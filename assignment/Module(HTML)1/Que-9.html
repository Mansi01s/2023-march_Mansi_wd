Ans:-Difference between Cell Padding and Cell Spacing

No. Cell Padding Cell Spacing
1. It specifies the space between the border of a table cell and
its contents.		 It specifies the space between adjacent cells.
2. It is created by using HTML <table> tag but type
attribute is set to cellpadding.		 It is also created by using HTML <table>tag but type attribute
is set to cellspacing.
3. It is mainly meant for a single cell. Cellspacing can get subjected to more than one cell.
4. The default cellpadding value is 1 Whereas, the default cellspacing value is 2
