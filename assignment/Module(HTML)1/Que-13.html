Quetion 13:-What is the use of an iframe tag?

Ans:-The <iframe> tag specifies an inline frame.
   An inline frame is used to embed another document within the current HTML document.
  They are commonly used for advertisements, embedded videos, web analytics and interactive content.

Iframe Advantages:

  1. easy to use
  2. well supported
  3. allows embedding of third-party content

Iframe Disadvantages:

  1. memory and processor heavy
  2. no control of third-party sites
  3. difficult to use in responsive design
