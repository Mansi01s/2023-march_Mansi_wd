Quetion 11:-What is the difference between a block-level element and an inline element?

Ans:-Difference between a block-level element and an inline element

No. Block-Level Elements Inline Elements
1. Block Elements occupy the full width irrespective
of their sufficiency.		 Inline elements occupy only sufficient width required.
2. Block elements always start in a line. Inline elements don’t start in a new line.
3. Block elements have top and bottom margin. Inline elements don’t have top and bottom margin

Example:-  Block-Level Element

Mansi Savaliya

The P and the DIV elements are both block elements, and they will always start on a new line and take up the full width available

Example:-  Inline Element

Mansi Savaliya

The SPAN element is an inline element, and will not start on a new line and only takes up as much width as necessary.
