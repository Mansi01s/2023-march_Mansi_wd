Quetion 8:-What are the various formatting tags in HTML?

Ans:-Various formatting tags:

1.<Bold> and <strong> Tags: Both tags are used to make the text bold. The text content of the tag is shown as important information on the webpage.

Syntax:

  • <b>.....</b>
  • <strong>.....</strong>

2.<Italic> and <Emphasized> Tags: Both tags are used to make the text italic and emphasized. Both the element have opening and closing tags.

Example:

  1. Italic Tag
  2. Emphasized Tag

3.<small> and <big> Tags: The <small> tag is used to set small font-size where as <big> tag is used to set big font-size.

Example:

  • Small Tag
  • Big Tag

4.<sup> and <sub> Tags: The <sup> tag is used to superscript a text whereas <sub> tag is used to subscript a text.

Example:

  • h2o 
  • 217

5.<ins> and <del> Tag: The <ins> tag is used to underline a text marking the part as inserted or added. This tag is mainly used in text in place of deleted text whereas <del> tag is used to delete the text it adds a strike line on the text.

Example:

  • Abcd
  • Abcd

6.HTML <mark> Tag: The <mark> tag is used to highlighting a text. It has an opening and closing tag.
