Quetion 12:-How to create a Hyperlink in HTML?

Ans:-The anchor tag <a> is used to create a hyperlink in HTML.

Syntax: <a href=" ">....</a>

There are three different types of hyperlinks on the web.

  1. relative
  2. absolute
  3. Inline
