Quetion 1:- What are the new tags added in HTML5?

Ans:- There are many new tags added in HTML5.

Example:

  (1) <section>: The section element represents a generic section of a document applicationA section in this context is a thematic grouping content typically with a heading.
  (2) <aside>: The aside element represents a section of a page that consists of content that is tangentially related to the content around the aside element.
  (3) <artical>: This article element represents a self- contained composition in a document, page, application, or site.
  (4) <audio>: Defines an audio file.
  (5) <video>: Defines an video file.
  (6) <embed>: The embed element represents an integration point for an external application or interactive content.
  (7) <hgroup>: The hgroup element represents the heading of a section.
  (8) <meter>: - The meter element represents a scalar measurement within a known range or a fractional value.
  (9) <progress>: The progress element represents the completion progress of a task.
  (10) <time>: The time element represents either a time on a 24 hour clock or a precise date in the proleptic gregorian calendar, optionally with a time and a time zone offset.
  (11) <Nav>: The nav element represents a section of a page that links to other pages or to parts within the page a section with navigation links.
  (12) <figure>: The figure element represents some flow content, optionally with a caption that is self contained and is typically referenced as a single unit from the main flow of the document.
  (13) <canvas>: Providing scripts with a resolution dependent bitmap canvas, which can be used for rendering graphs, game graphics, or other visual images on the fly.
