Quetion 1:- What are the new tags added in HTML5?

Ans:- There are many new tags added in HTML5.

Example:

  (1) <section>: Represents a generic document or application section.
  (2) <aside>: Represents a piece of content that is only slightly related to the rest of the page.
  (3) <artical>: Represents an independent piece of content of a document, such as a blog entry or newspaper article.
  (4) <audio>: Defines an audio file.
  (5) <video>: Defines an video file.
  (6) <embed>: Defines external interactive content or plugin.
  (7) <hgroup>: Represents the header of a section.
  (8) <meter>: Represents a measurement, such as disk usage.
  (9) <progress>: represents the completion progress of a task.
  (10) <time>: Represents a date and/or time.
  (11) <Ruby>: Together with <rt> and <rp> allow for marking up ruby annotations.
  (12) <figure>: Represents a piece of self-contained flow content, typically referenced as a single unit from the main flow of the document.
