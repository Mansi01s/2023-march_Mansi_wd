Quetion 2:- How to embed audio and video in a webpage?

Ans:- The HTML5 <audio> and <video> tags make it simple to add media to a website.Then set src attribute to identify the media source and include a controls attribute so the user can play and pause the media.

Syntax:

(1) Audio Embed:
  <audio controls>
   <source src=" " type=" ">
  </audio>

(2) Video Embed:
  <video controls>
   <source src=" " type=" ">
  </video>
