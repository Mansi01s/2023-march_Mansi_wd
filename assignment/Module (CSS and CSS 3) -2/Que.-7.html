Quetion 7:- How can image repetition of the backup be controlled?

Ans:-
Control the repetition of an image in the background, use the background-repeat property.The background-repeat property is used for repeat the background image both horizontally and vertically.It also decides whether the background image will be repeated or not.

Example:

1. background-repeat:  repeat;
2. background-repeat:  repeat-x;
3. background-repeat:  repeat-y;
4. background-repeat:  no-repeat;
