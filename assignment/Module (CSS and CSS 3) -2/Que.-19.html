Quetion 19:- What is the rule set?

Ans:- A rule set also called rule.A rule set in CSS is a collection of style declarations.There are more than 100 different properties in CSS and a nearly infinite number of different values.

There are some different properties:

(1) Selector

(2) Declarations

(3) Element Selector

(4) IdSelector

(5) class Selector

(6) values
