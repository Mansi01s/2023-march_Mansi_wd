Quetion 13:- What are the ways to integrate CSS as a web page?

Ans:- There are 3 ways to integrate CSS as a web page.

  1. Inline - by using the style attribute inside HTML elements.
  2. Internal - by using a <style> element in the <head> section.
  3. External - by using a <link> element to link to an external CSS file.

Example:-

  (1) Inline CSS:
   <span style=""></span>

  (2) Internal CSS:

  <html>
   <head>
     <title></title>
     <style></style>
   </head>
   <body>
      <div class="box"></div>
   </body>
   </html>

  (3) External CSS:
  <html>
   <head>
     <title></title>
     <link rel="stylesheet" href="style.css">
   </head>
   <body>
      <div></div>
   </body>
   </html>
