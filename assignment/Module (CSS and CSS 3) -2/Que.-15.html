Quetion 15:- What are the external style sheets?

Ans:- An external style sheet is a separate CSS file that can be accessed by creating a link within the head section of the webpage. External CSS used to add styling to multiple HTML pages at a time.It helps to design the layout of many        HTML web pages same. The external CSS is always saved with the ".css" extension.

Example:-

  <!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title></title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
     <div></div>
  </body>
  </html>
