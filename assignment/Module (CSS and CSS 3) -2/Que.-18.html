Quetion 18:- What are the media types allowed by CSS?

Ans:- There are few media types allowed by CSS.

(1) All - Used for all media type devices.
(2) print - Used for printers.
(3) screen - Used primarily for color computer screens.
(4) tv - Used for television-type devices - low resolution, color, limited-scrollability screens, sound available.
(5) projection - Used for projected presentations, for example projectors.
