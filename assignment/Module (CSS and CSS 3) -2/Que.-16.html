Quetion 16:- What are the advantages and disadvantages of using external style sheets?

Ans:- The advantages and disadvantages of using external style sheets

Advantages:

  • The styles of multiple documents can be controlled from one file.
  • Classes can be created for use on multiple HTML element types in many documents.
  • one change to the style sheet will change all linked pages

disadvantages:

  • The external style sheet is loaded, it may not be possible to render the document.
  • For small number of style definitions, it is not viable.
  • Extra Work for Developers.
