Quetion 14:- What is embedded style sheets?

Ans:- Embedded style sheets is a embedded CSS. It allows you to define styles for a particular HTML document as a whole in one place. his is done by embedding the <style></style> tags containing the CSS properties in the head of your document.

Example:

  <html>
   <head>
     <title></title>
     <style>
     </style>
   </head>
   <body>
      <div class="box"></div>
   </body>
   </html>
